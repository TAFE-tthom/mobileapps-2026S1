
# Command Line Cheat Sheet

## Filesystem Interaction

* `cd` - Change Directory, allows you to change the directory you are currently in, this accepts relative and absolute pathing.

* `ls` - List directory, allows you to list the contents of a directory.

* `mkdir` - Make directory, allows you to create a directory

* `mv` - Move, this command also allows you to rename the file

* `rm` - Remove, this allows you to remove the file itself

* `cp` - Copy, this allows you to copy the file or folder (with a flag)

* `pwd` - Prints the current working directory (where you are currently)

* `rmdir` - Removes a directory specified, must be empty

## File/Stream Interaction

* `cat` - Reads a file specified as an argument

* `head` - Reads the top N lines of a file or stream

* `tail` - Read the last N lines of a file or stream




## Keyboard Shortcuts

* `CTRL + a` - Goes to the beginning of a line of the buffer you are using

* `CTRL + e` - Goes to the end of a line of the buffer you are using

* `CTRL + w` - Deletes the previous word

* `CTRL + c` - Interrupts the current process running

* `CTRL + d` - Deletes a character forward OR if a process is in context, sends end of input.
